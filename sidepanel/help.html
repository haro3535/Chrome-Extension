Dino Marker Extension Guide


Welcome to Dino Marker, your simple and efficient text highlighter for Chrome! Below is a step-by-step guide on how to use this extension:



1. Home, Settings, and Help Icons (Top Bar)


At the top of the side panel, you'll see three icons:

  • ( Home ) Takes you to the main dashboard of the extension.

  • ( Settings ) Customize advanced options for the highlighter.

  • ( Help ) Access helpful tips and guides on how to use Dino Marker.



2. Selection Mode Tabs (Main Toolbar)


Just below the top bar, you’ll find three tabs representing different modes for highlighting or interacting with the text:

  • ( Arrow Mode ) Switch to arrow mode to select or navigate without making any highlights.

  • ( Highlight Mode ) Enable this mode to highlight text on a webpage. Simply click and drag over any text you'd like to mark.

  • ( Erase Mode ) Use this to erase any highlighted text that you've previously marked.



3. Adjust Opacity


0.8

Opacity Slider: Adjust how transparent or bold you want your highlights to appear. Move the slider to the left for more transparency or to the right for more opacity. The current opacity is shown next to the slider.



4. Choose Your Marker Color



Color Picker: Select one of the six available colors (Yellow, Red, Orange, Blue, Green, Purple, Pink) to highlight your text. Simply click on a color, and it will be used for all future highlights until you change it. The selected color will have a blue border, indicating that it is currently active.



5. Marked Texts Section


Marked Texts: This area will display the list of texts you have highlighted on the current page or across sessions. Currently, this feature is under development and will soon display your marked texts for easy reference.

Note: The "Coming Soon..." label indicates that this feature will be available in a future update.



6. Footer


At the bottom of the side panel, you’ll find the version information for the Dino Marker extension. This ensures you are using the latest version and can track updates easily.



That’s it! Now you’re ready to start marking up the web with Dino Marker. If you have any further questions, just click the Help icon for additional assistance.


Enjoy highlighting!
